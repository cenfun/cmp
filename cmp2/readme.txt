
CenFun Music Player v2.0 b060706

	――适合于DIY站长使用的Flash音乐播放器
=========================================
升级说明：
新增歌词/列表的自定义颜色配置，是否自动播放配置
新增播放进度条点击跳转功能
其他细节优化，等等
升级方法：
1，覆盖新版cfplay.swf
2，覆盖amdin文件夹中的新版manage.asp (首次运行先进行一次播放器配置即可)
=========================================
文件说明:
admin  管理目录
bg  背景图片目录
lrc  歌词目录
scene  场景目录
xml  配置文件和歌曲列表目录
=========================================
基本运行环境:
Win2000或更高+IIS5.0或更高+IE5.0以上+Access数据库支持
文件目录需要有写入权限,支持FSO
(如果你的网站空间不支持这些，可先在本地运行服务管理完或者手动编辑xml列表，然后只需要上传xml文件夹中的文件即可)
=========================================
管理登录：
默认管理登录入口：admin/login.asp
默认管理员用户名密码都为：admin 
=========================================
■安全设置■(很重要,务必进行修改)：
1,请在第一次登录后进行用户名和密码修改！
2,对数据库进行更名，默认位置：admin/data/#cfplay_data_2006.mdb
更名后请务必对数据库连接文件admin/conn.asp第15行进行对应修改：Db = "data/#cfplay_data_2006.mdb" 
推荐在数据库文件名中加#符号，防止恶意下载，并将后缀mdb改为asp、asa等等
3,更改管理入口文件夹名，比如将admin文件夹改为admin2008或者manage等等
此操作不影响文件生成，只要管理文件夹和cfplay.swf处于同级目录即可
