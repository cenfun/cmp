    CMP是一款免费的在线音乐播放器，采用Flash开发。
支持的音频视频格式主要有MP3,FLV,MP4,并可以通过网页
JS接口实现WMP(Windows Media Player)所支持的格式播放，
如wma,wmv,wav,mid等(仅声音)。主要功能特色有列表式播
放(自定义音乐专辑和列表)，MP3可视化效果(SoundMixer
混音器)，同步歌词，高清视频(h264)，自定义皮肤，加载
各种插件等等。

上传说明：请上传upload目录中的文件到你的服务器:
----------------------------------------------------
cmp.swf          播放器主程序
config.xml       配置文件
list.xml         列表文件
index.htm        首页
cmp.js           Flash插入脚本
crossdomain.xml  跨域策略文件
skins            皮肤目录
plugins          插件目录
lrc              歌词目录
注：如果MP3在你另外的空间(不同域)，请将crossdomain.xml
和MP3文件放在一起，或域的根目录，即可实现可视化效果
----------------------------------------------------

文件TrustFiles.txt为去除Flash本地安全限制补丁
请将其拷贝到以下目录中即可：
C:\WINDOWS\system32\Macromed\Flash\FlashPlayerTrust\
其中C为系统盘，请根据你的情况对号入座
如果FlashPlayerTrust不存在就新建一个该名称的目录

附：快捷键汇总
----------------------------------------------------
回车：播放当前选中曲目
空格：播放暂停切换
方向键上：上移动列表项
方向键下：下移动列表项
方向键左：移动到分类列表
方向键右：移动到音乐列表
ctrl+回车：视频最大化切换
ctrl+方向键左：快退
ctrl+方向键右：快进
ctrl+alt+方向键左：前滚选择混音效果
ctrl+alt+方向键右：后滚选择混音效果
ctrl+数字(0-9)：快速切换皮肤ID
F键：全屏切换
D键：恢复系统默认皮肤
R键：重载或刷新当前皮肤
O键：打开或关闭选项窗口
L键：打开或关闭列表窗口
C键：打开或关闭歌词窗口
V键：打开或关闭视频窗口
P键：开启或禁用所有插件
